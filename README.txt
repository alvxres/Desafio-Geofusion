===========INSTRUÇÕES PARA EXECUTAR O BOT (ROBÔ):===============

Abrir o arquivo "Bot.jar" em uma pasta contendo chromedriver.exe. (Caso este esteja faltando, o bot não fará nada).


===================O QUE O BOT FARÁ:========================

Abrirá o Chrome de forma automatizada, carregará a página em questão, selecionando a opção "CAMPINAS" e salvará os dados no arquivo "Dados.csv" 
(que deve se encontrar no mesmo local o qual o bot fora aberto). Logo após, o Chrome será fechado. Tarefa finalizada.
